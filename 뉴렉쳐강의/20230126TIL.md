* ROWNUM은 WHERE절이 실행되면서 만들어진다.
```
SELECT * FROM (
    SELECT ROWNUM NUM, N.*
    FROM (SELECT * FROM NOTICE ORDER BY REGDATE DESC) N
) 
WHERE NUM BETWEEN 6 AND 10;
```
